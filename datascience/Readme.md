❤️ Heart Disease Prediction Modeling
📌 Overview

This project focuses on predicting the likelihood of heart disease using patient health data and machine learning models. By analyzing key medical attributes, the model helps identify high-risk patients and supports data-driven healthcare decision-making.

🎯 Objective
Identify key predictors contributing to heart disease
Build a classification model to predict patient risk
Improve interpretability for healthcare insights

📊 Dataset Overview

The dataset includes demographic and clinical features such as:

Age, Sex
Cholesterol levels
Resting blood pressure
Maximum heart rate (thalach)
Chest pain type
Exercise-induced angina
ST depression (oldpeak)
Number of major vessels (ca)
Target (heart disease presence)

🧠 Approach
🔍 Exploratory Data Analysis (EDA)
Analyzed correlations between health indicators and heart disease
Identified key predictors such as age, cholesterol, and heart rate
Visualized distributions and relationships using plots

🧹 Data Preprocessing
Handled missing values and inconsistencies
Normalized/standardized numerical features
Encoded categorical variables

⚙️ Model Development

Built and compared multiple classification models:

Logistic Regression
Random Forest
K-Nearest Neighbors (KNN)

📏 Evaluation Metrics
Accuracy (>85%)
Precision & Recall
F1 Score
ROC-AUC

📈 Results
Achieved over 85% accuracy across models
Random Forest provided strong performance with better feature importance insights
Identified critical health indicators influencing heart disease risk

📊 Visualization
Used matplotlib and seaborn to:
Plot correlations
Compare model performance
Communicate insights effectively

💡 Key Insights
Higher cholesterol and abnormal heart rate significantly increase risk
Age is a strong contributing factor
Certain chest pain types are strong indicators of heart disease

💼 Business Impact
Enables early detection of heart disease
Supports doctors in making informed clinical decisions
Helps prioritize high-risk patients
Improves overall healthcare efficiency
