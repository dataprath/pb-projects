📊 Best Cities Dashboard (Excel)

📌 Overview
This project presents an interactive Best Cities/States Dashboard built in Microsoft Excel by combining data from multiple public sources. The goal was to evaluate and compare U.S. states across safety, education, crime, and demographic factors to identify overall performance patterns.

The dashboard enables users to explore insights on city performance using dynamic filters and visualizations.

🎯 Objective
To analyze and rank cities based on multiple criteria such as cost of living, quality of life, and economic indicators, and present the findings through a user-friendly dashboard.

🗂️ Data Sources

The dashboard integrates data from the following sources:

Safety Score by State – [World Population Review](https://worldpopulationreview.com/state-rankings/safest-states)
Top Colleges Ranking – [Forbes](https://www.forbes.com/top-colleges/)
University Rankings – [4ICU](https://www.4icu.org/us/)
Crime Rate Index – [USA.com](http://www.usa.com/rank/us--crime-index--state-rank.htm?hl=&hlst=&wist=&yr=&dis=&sb=ASC&plow=&phigh=&ps=)
Political Affiliation by State – [Pew Research Center](https://www.pewresearch.org/religious-landscape-study/database/compare/party-affiliation/by/state/)

🛠️ Tools & Techniques Used
Microsoft Excel
Power Query (data extraction & transformation)
Pivot Tables & Pivot Charts
Data Modeling
Slicers for interactivity
Conditional Formatting

🔄 Data Preparation
Collected and consolidated datasets from multiple sources
Standardized state names and formats for accurate merging
Cleaned missing and inconsistent values
Created calculated metrics to align different ranking systems
Built relationships across datasets using state-level keys

📈 Dashboard Features
Interactive filters (state-level selection)
Comparative analysis across safety, crime, and education
Dynamic ranking visuals
KPI indicators for quick insights
Multi-source data integration in a single view

💡 Key Insights
States with higher safety scores tend to have lower crime index values
Variations in education rankings highlight regional differences
Political and demographic factors provide additional context to state performance

🚀 How to Use
Open the Excel dashboard file
Use slicers to filter by state or metric
Explore charts and KPIs to compare performance
